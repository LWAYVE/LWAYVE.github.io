Set the preferred language for audio clips and associated metadata in the SDK. (optional) If not set the SDK will use will attempt to use the device language if it is supported, otherwise it will fallback to the default language. The language used for audio content and associated metadata See ExperienceLanguage enum for list of languages supported by the SDK.
